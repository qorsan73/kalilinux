# kalilinux
A Kali Linux termx conversion tool with an area not exceeding 50 MB

# Specifications 
 💾 Total Size: 48MB (Compressed)
🛠️ Tools: 150+ Penetration Testing Tools
📱 Compatible: Termux ARM64 (Android)
👑 No Root Required
⚡ Fast Startup: 10 Seconds 
✅ Compressed to only 48MB (instead of the regular 2GB)
✅ Automatically installs all essential tools
✅ Built-in password lists
✅ MSF Database ready
✅ Organized folders for pentesting
✅ No reinstallation required 

# Included tools 
🌐 Survey: nmap, masscan, dnsrecon, sublist3r
💉 Exploit: metasploit, sqlmap, beef-xss
🔑 Cracking: hydra, hashcat, john
🕸️ Web: nikto, dirsearch, gobuster, ffuf
📡 Wireless: aircrack-ng, wifite, reaver
